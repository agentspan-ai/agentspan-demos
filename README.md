# agentspan-demos

Small, runnable Agentspan demos and companion notebooks.

## Notebook demos

- [End-to-End Agentspan System Design Demo](./notebooks/api-tool-openapi-service/): a short notebook that shows one complete Agentspan workflow with an external integration, custom Python tools, multiple agents, a built-in multi-agent strategy, deploy/serve/run, and UI monitoring.
- [API Tool OpenAPI Service Demo](./notebooks/api-tool-openapi-service/): a notebook that shows how to build an Agentspan agent on top of an existing OpenAPI-described API.

Both notebooks currently live in the same directory because they share the same mock service and supporting files.
